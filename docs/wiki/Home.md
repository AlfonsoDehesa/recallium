# Recollectium Wiki

Welcome to the Recollectium manual.

Recollectium is a local-first memory engine for AI agents. It gives agents a durable, inspectable place to store and search facts, preferences, decisions, task context, and project knowledge without locking that memory inside one client.

- [Installation](Installation): install methods, upgrade, uninstall, and shell completion.
- [Concepts](Concepts): user memory, workspace memory, memory buckets, embeddings, and workspace UIDs.
- [Configuration](Configuration): config file location, defaults, CLI overrides, and directory paths.
- [Features and Commands](Features-and-Commands): what Recollectium can do and where each feature lives.
- [CLI Reference](CLI-Reference): command-by-command reference.
- [Service Management](Service-Management): managed API and MCP services, discovery, PID files, and foreground serve mode.
